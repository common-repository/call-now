== Description ==

###What does the plugin do?

Live Agent Call is the most simplest help desk plugin for wordpress website. provide platform to stay closer to site customers and answer their questions faster with Phone call.  

###Could not be easier!
The settings are very easy: enable and enter your sip information. That's it!

But if you want more control, you can always open up the Advanced Settings and change the default behavior.



== Installation ==

1. From your WordPress Dashboard go to 'Plugins' > 'Add new' and search for 'Call Now'.
2. Click 'Install Now' under the title of the Call Now plugin
3. Click activate
4. Go to 'Settings' > 'Call Now' and check the box "Enable Call Now" for call button and enter text your phone required fields.
5. Click 'Save' and you're done!

Or:

1. Upload `Call Now`-folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings' > 'Call Now' and check the box "Enable Call Now" for call button and enter text your phone required fields.
4. Click 'Save' and you're done!
